jFSTMerge Execution Steps


Introduction


In this page, we will provide an overview of the scripts results obtained from using the jFSTMerge tool, along with important information to understand and reproduce these results.

Overview

We executed jFSTMerge over our subjects using shell script, then we preprocessed the results using Python scripts.

How to Reproduce

If you wish to reproduce the results with the jFSTMerge tool, follow these steps:

  1. Prerequisites: To obtain jFSTMerge Tool simulation results:
    • Windows and Java 8 (Java version "1.8.0_212" or above) to execute jFSTMerge
    • Git (optional)
    • A list of merges containing merge parent 1 (p1), merge parent 2 (p2), and the merge commit.
  2. To analyze the jFSTMerge simulation results:
    • Python
    • Pandas and relationed libraries
  3. Clone the jFSTMerge Tool repository:
    git https://github.com/guilhermejccavalcanti/s3m.git
  4. Run the script to execute jFSTMerge:
    .\jfst_script.sh
  5. Collect results from jFSTMerge execution using the scripts: arquivo com preprocessamento da jfstmerge.ipynb
