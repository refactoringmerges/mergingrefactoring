IntelliMerge Execution Steps


Introduction


In this page, we will provide an overview of the scripts results obtained from using the IntelliMerge tool, along with important information to understand and reproduce these results.

Overview

We executed IntelliMerge over our subjects using shell script, then we preprocessed the results using Python scripts.

How to Reproduce

If you wish to reproduce the results with the IntelliMerge tool, follow these steps:

  1. Prerequisites: To obtain IntelliMerge Tool simulation results:
    • Windows and Java 8 to execute IntelliMerge
    • A list of merges containing merge parent 1 (p1), merge parent 2 (p2), and the merge commit.
  2. To analyze the IntelliMerge simulation results:
    • Python
    • Pandas and relationed libraries
  3. Clone the IntelliMerge Tool repository:
    git https://github.com/Symbolk/IntelliMerge.git
  4. Run the script to execute IntelliMerge:
    .\intellimerge_script.sh
  5. Collect results from IntelliMerge execution using the scripts: IntelliMerge_preprocessing.ipynb
